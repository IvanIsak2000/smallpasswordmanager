# smallpasswordmanager

## table of commands:

| command                   	| description              	|
|---------------------------	|--------------------------	|
| ``` python3 main.py a ``` 	| append password          	|
| ``` python3 main.py b ``` 	| delete password          	|
| ``` python3 main.py r ``` 	| read passwords (default) 	|
